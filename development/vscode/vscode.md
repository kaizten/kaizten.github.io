# Introducción a Visual Studio Code (VSCode)

Visual Studio Code (VSCode) es un editor de código fuente desarrollado por Microsoft. Es ligero, rápido y altamente extensible, lo que lo convierte en una herramienta ideal para desarrolladores de todos los niveles. VSCode soporta múltiples lenguajes de programación y ofrece características como depuración integrada, control de versiones, y una amplia gama de extensiones para personalizar tu entorno de desarrollo.

## Activar AutoSave de ficheros

AutoSave es una funcionalidad que guarda automáticamente los cambios realizados en tus archivos, evitando la pérdida de progreso. Para activar AutoSave:

1. Abre VSCode.
2. Ve al menú superior y selecciona `File` (Archivo).
3. Haz clic en `Auto Save` (Guardado Automático).
4. Alternativamente, puedes configurar el intervalo de guardado automático en la configuración de usuario (`File > Preferences > Settings` o `Ctrl + ,`) buscando "Auto Save" y seleccionando una opción como `afterDelay` o `onFocusChange`.

## Buscar archivos en VSCode

VSCode facilita la búsqueda de archivos dentro de tu proyecto. Para buscar archivos:

1. Presiona `Ctrl + P` (o `Cmd + P` en macOS) para abrir la barra de búsqueda rápida.
2. Escribe el nombre del archivo que deseas encontrar. VSCode mostrará una lista de coincidencias en tiempo real.
3. Selecciona el archivo deseado para abrirlo.

Además, puedes usar la barra lateral de Explorador (`Ctrl + Shift + E`) para navegar por la estructura de carpetas y archivos de tu proyecto.

Con estas herramientas, trabajar en proyectos grandes y complejos se vuelve más eficiente y organizado.
